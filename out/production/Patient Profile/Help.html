Patient Management Application - City of Laredo Health Department

Getting Started

  1. Clone or download the application from the GitHub repository.
  2. Make sure that you have the latest version of Java installed on your computer.
  3. Open the project in an IDE of your choice (such as Eclipse, NetBeans, or IntelliJ IDEA).
  4. Run the application.

Features

The application has three main tabs:

  • View Patient - This tab allows the user to search for a patient by patient ID, name, and date of birth. Once selected, an individual patient profile with their information is displayed.
  • Dashboard - This tab displays visual statistical graphs about the patient data set. The graphs give insights into patient demographics, medical conditions, and other relevant information.
  • Insert Patient - This tab allows the user to insert new patient information into the original .csv file. The user can enter details such as patient ID, name, date of birth, and other general information.

How to Use

To use the application, follow these steps:

  1. Open the application.
  2. Select the desired tab based on the task that you want to perform.
  3. If you want to search for a patient, enter their patient ID, name, or date of birth in the search box on the "View Patient" tab and click the "Search" button.
  4. If you want to insert a new patient, go to the "Insert Patient" tab and enter the required information in the input fields.
  5. If you want to view statistical graphs about the patient data set, go to the "Dashboard" tab.
  6. After completing your desired task, you can exit the application by clicking the "Exit" button.
